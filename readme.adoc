It is mainly used to use class name prefixes to control CSS styles under class names to *#prevent style pollution#*ã€‚

== Usage

In index.js, modify *#targetFilePath#* and *#pre#* to your own.

====
```js
include::./index.js[]
```
====

== example

.input
====
```css
.test{
    color:red
}
```
====

.output
====
```css
.adoc .test{
    color:red
}
```
====

== limitations

Now it can not handle media query which *#block is nested#*, so it can not handle scss or sass.
